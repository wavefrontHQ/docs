Wavefront proxies support the following data formats: - [Wavefront data format](wavefront_data_format.html) - [Graphite data format (plaintext and pickle)](http://graphite.readthedocs.io/en/latest/feeding-carbon.html) - [OpenTSDB data format (Telnet interface and HTTP API (JSON))](http://opentsdb.net/docs/build/html/user_guide/) Before streaming data, we recommend that you understand [best practices for naming your data](wavefront_data_format.html#data-format-best-practices).
